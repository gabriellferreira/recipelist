# Recipelist

Project based in the MVVM clean architecture.

* minSdkVersion: 21
* targetSdkVersion: 29

## Compile
To compile this project, add tags below to your global 'gradle.properties':
* API_SPACE_ID
* API_ENVIRONMENT_ID 
* API_ACCESS_TOKEN 

## Dependencies:
* Kotlin
* AndroidX
* LiveData
* ViewModel
* Retrofit
* Dagger2
* RxJava
* Glide
* Gson
* DetektCheck
* JUnit 4
* Mockito
